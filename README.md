# Smart Grocery Billing System

A simple Python-based billing application that allows users to enter grocery items, validates product availability, calculates quantities, generates an invoice, and displays the total bill.

## Features

* Product validation
* Quantity calculation
* Automatic bill generation
* Invalid item detection
* User-friendly console output

## Technologies Used

* Python
* Dictionaries
* Lists
* Loops
* Conditional Statements

## Run the Program

```bash
python smart_grocery_billing_system.py
```

## Sample Input

```text
apple, milk, apple, eggs
```

## Sample Output

```text
Apple x2 = ₹100
Milk x1 = ₹30
Eggs x1 = ₹60

Total Bill = ₹190
```
